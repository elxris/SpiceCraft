#Useless Plugin
--------

##¿Qué hace este plugin?
Tiene tre funciones principalmente.

### Mail
Esta es la función más importante.
Permite a los jugadores, enviar mensajes entre sí, ya sea si están conectados o no.
Permite al administrador hacer un envío masivo de correos.
La ayuda está en el juego mediante el comando `/mail`
### Warps temporales
Puedes hacer warps temporales personales o generales, los cuales pueden ser accdedidos por cualquier persona.
Funciona entre mundos.
Ayuda mediante `/tw`
### Compás
Hace que el compás apunte a una dirección que haya definido el usuario.
Ayuda mediante `/upin`
