#Useless Plugin
--------

##¿Qué hace este plugin?
Es un plugin sin temática definida, por tanto tiene varias funciones.
Mail, Compás, Warps Temporales, Tienda Virtual

### Mail
Esta es la función más importante.
Permite a los jugadores, enviar mensajes entre sí, ya sea si están conectados o no.
Permite al administrador hacer un envío masivo de correos.
La ayuda está en el juego mediante el comando `/mbox`
### Warps temporales
Puedes hacer warps temporales personales o generales, los cuales pueden ser accedidos por cualquier persona.
Funciona entre mundos.
Ayuda mediante `/tw`
### Compás
Hace que el compás apunte a una dirección que haya definido el usuario.
Ayuda mediante `/upin`
### Comandos
El servidor puede definir comandos para ayuda dentro del servidor por ejemplo. Tiene una flexible funcionalidad, ya que puedes definir los comandos como tu desees, además de poner los subcomandos que quieras, con la profundidad que quieras. Un ejemplo de como configurarlo se encuentra en el archivo generado: `commands.yml`
### Tienda
Es una de las funciones importantes del comando, ya que permite una innovadora tienda virtual con precios que varían. Estos precios están determinados por la cantidad de ventas y de compras. Da una sensación de una economía variante y fluida.
